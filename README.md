## Installation

GIFFromImages is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'GIFFromImages'
:git => 'https://github.com/Murmeko/GIFFromImages.git'
```
